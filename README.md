# Moniteur de Ressources Système

Un outil de monitoring en temps réel des performances de l'ordinateur.

## Fonctionnalités

- **Surveillance en temps réel** : RAM et CPU
- **Graphiques interactifs** : Fluctuations des performances
- **Sauvegarde des données** : Compte utilisateur avec historique
- **Export des sauvegardes** : Format JSON
- **Interface moderne** : Design sombre professionnel

## Technologies utilisées

- **HTML5** : Structure de la page
- **CSS3** : Styles et design
- **JavaScript** : Logique et interactivité
- **Chart.js** : Graphiques en temps réel
- **Font Awesome** : Icônes professionnelles
